# BTTS Target Tree Pack

**Objetivo do pack**
Preparar a árvore técnica-alvo do módulo BTTS para futura migração controlada, mantendo o estado atual em teste enquanto a camada de API/dados ainda está a ser estabilizada.

**Ficheiros principais**
- `TARGET_TREE_BTTS.md`
- `INTEGRATION_NOTES_BTTS.md`
- `EXPORT_CONTRACT_BTTS.md`
- `DATA_HANDOFF_BTTS.md`

**Dependências**
- Contrato Transversal de Integração SignalBet v1.1 Operacional
- Motor BTTS v1 com BOS, BVS, SBI, AMI, FGT, TSI, xG Gap e penalizações finais
- Data/API Layer central em evolução

**Destino final pretendido**
`modules/btts/`

**Estado**
`teste`
