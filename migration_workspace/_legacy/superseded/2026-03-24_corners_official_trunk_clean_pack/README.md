# Corners Official Trunk Clean Pack

## objetivo
Versão limpa do pack de Corners ligada ao Data_API_Official_Trunk_v1.

## provider oficial
- Data_API_Official_Trunk_v1

## serviços consumidos
- get_fixtures_by_league_season()
- get_fixture_statistics()

## fluxo curto
1. leitura do trunk
2. adaptação para o motor
3. geração de candidate pick
4. output contratual

## objetivo analítico
- 1 forte = candidate
- 1 médio = watchlist
- 1 rejeitado = rejected
