# Failure Policy

## Bloqueia validation run
- falta `api.env.txt`
- falta `API_KEY`
- falta `data_api/`
- falta `Data_API_Official_Trunk_v1`
- providers mínimos ausentes no tronco
- contracts mínimos ausentes no tronco

## Permite continuar com warning
- snapshot/cache ausente
- bridges centrais em placeholder
- módulos disabled por maturidade
