# UI Frontend SignalBet v25 — Consolidation Governance Pack

## Objective
Consolidate the protected bridge as a controlled, repeatable, and legible integration layer without pretending full live integration.

## Focus of this pack
- operational stability matrix
- harder bridge decision policy
- clear separation between protected real source and useful simulation
- improved operational readability of the “Pôr tudo a correr” panel
- short phase-boundary status note

## Current status
This pack remains in **staging**.
It does **not** implement full live runtime consumption.
It does strengthen the UI’s governance and legibility around:
- reuse
- refresh
- fallback
- requested_mode vs observed_mode
- stable vs experimental behavior
