# UI v25 — Orchestrator Panel Readability Focus

The panel should prioritize readability over field accumulation.

## Fields to emphasize
- requested_mode
- observed_mode
- fallback_used
- bridge_status
- bridge_scope
- stability_status
- source_transition
- bridge_decision_reason

## Readability rule
These fields should be shown with clear grouping:
1. Source intent
2. Source actually used
3. Bridge health
4. Stability / consistency
5. Decision reason
