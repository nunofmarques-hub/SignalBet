# Handoff Notes

Este pack fecha quatro objetivos do relatório do chefe:
1. adapter explícito à Data/API Layer;
2. exemplos com inputs centrais realistas;
3. output oficial do módulo Cards no contrato v1.1;
4. critérios de elegibilidade mais fechados com testes de ponta curta.

## Limites desta entrega
- Os inputs são realistas e estruturados como payloads centrais, mas continuam a ser exemplos de staging.
- O scoring é v1 e ainda não representa maturidade analítica final.
- O adapter já mostra claramente o handoff de input esperado para evolução posterior.
