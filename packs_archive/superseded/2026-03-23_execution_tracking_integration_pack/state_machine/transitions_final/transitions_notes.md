# State transitions finais v1

## Fluxo principal
- RECEIVED -> VALIDATION_PENDING
- VALIDATION_PENDING -> VALIDATED
- VALIDATED -> READY_TO_EXECUTE
- READY_TO_EXECUTE -> EXECUTION_PENDING
- EXECUTION_PENDING -> EXECUTED
- EXECUTED -> AWAITING_SETTLEMENT
- AWAITING_SETTLEMENT -> SETTLED

## Fluxos alternativos
- VALIDATION_PENDING -> REJECTED_AT_EXECUTION
- EXECUTION_PENDING -> MISSED
- EXECUTION_PENDING -> CANCELLED
- EXECUTION_PENDING -> MANUAL_REVIEW
- MANUAL_REVIEW -> READY_TO_EXECUTE
- MANUAL_REVIEW -> MISSED
- MANUAL_REVIEW -> CANCELLED
- AWAITING_SETTLEMENT -> VOID

## Regra dura
Não é permitido saltar diretamente de RECEIVED para EXECUTED.

## Reason codes mais associados por fase
### validation
- INVALID_SCHEMA
- MISSING_REQUIRED_FIELD
- INVALID_SOURCE
- INVALID_DECISION_STATUS
- DUPLICATE_ORDER

### execution
- PRICE_DRIFT
- ODDS_OUTSIDE_WINDOW
- MARKET_SUSPENDED
- MARKET_CLOSED
- EXECUTION_TIMEOUT
- EXECUTION_NOT_ATTEMPTED
- DATA_MISMATCH

### settlement
- FIXTURE_CANCELLED
- FIXTURE_POSTPONED
- FIXTURE_ABANDONED
- RESULT_UNAVAILABLE
- SETTLEMENT_CONFLICT
- VOID_RESULT
