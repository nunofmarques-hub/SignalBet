# Shortlist and Batch Rules v2

## critérios mínimos de shortlist
- eligibility = true
- global_score_final >= 72
- priority_tier em Best / Top / Actionable
- sem bloqueio estrutural

## regra de fixture
Por defeito entra 1 pick por fixture.
A segunda pick do mesmo jogo deve ficar fora do lote final, salvo política futura explícita.

## batch multi-módulo
O batch deve aceitar picks de vários módulos e produzir:
- ranked batch
- shortlist batch
- export batch para banca

## buckets
- core: Best e Top
- support: Actionable
- watch: Watchlist/Reject fora do lote
